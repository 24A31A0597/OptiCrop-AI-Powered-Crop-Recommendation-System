# 🌾 OptiCrop – AI Powered Crop Recommendation System

## 📌 Project Overview

OptiCrop is an AI-powered Crop Recommendation System designed to assist farmers in selecting the most suitable crop based on soil nutrients and environmental conditions. The application uses a Machine Learning model to analyze agricultural parameters and predict the optimal crop for cultivation.

The project aims to improve agricultural productivity by providing accurate, data-driven crop recommendations through a simple and user-friendly web interface.

---

## 🚀 Features

- 🌱 Predicts the most suitable crop
- 📊 Uses Machine Learning for prediction
- 🌦️ Accepts soil and environmental parameters
- 💻 Interactive Flask web application
- 📱 Responsive user interface
- ⚡ Fast prediction with confidence score

---

## 📥 Input Parameters

The application accepts the following inputs:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

---

## 📤 Output

The system predicts:

- Recommended Crop
- Prediction Confidence

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/OptiCrop-AI-Powered-Crop-Recommendation-System.git
```

### Navigate to Project Folder

```bash
cd OptiCrop-AI-Powered-Crop-Recommendation-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

The application will start at:

```
http://127.0.0.1:5000
```
